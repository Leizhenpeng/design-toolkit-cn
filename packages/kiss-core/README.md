## Feature

- 🎁 专心业务逻辑，一套代码即可支持masterGo、jsDesign和Figma
- ⛽️ ts类型友好，支持自动补全
- ⭐️ 主动监测代码运行时环境，自由切换所处上下文
- 🍺 根据常用场景二次封装基础命令，加速开发
